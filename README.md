# Python仓库管理系统 python44

本项目是一款基于Python语言的仓库管理系统，集成了源码、运行教程以及开发环境，旨在为用户提供一个高效、便捷的仓库管理解决方案。

## 技术栈

- 后端：Python Django
- 前端：Vue3, Ant-design-vue
- 开发工具：PyCharm
- 后端构建工具：Pip
- 前端构建工具：Vite

## 运行环境

- 操作系统：Windows
- Python版本：3.8
- NodeJs版本：v16
- 数据库：MySQL 8.0

## 功能模块

- 商品管理
  - 商品信息维护
  - 商品库存管理
- 商品分类管理
  - 分类信息维护
- 用户管理
  - 用户权限管理
  - 用户行为跟踪

## 系统角色

- 系统管理员
- 商品管理人员
- 用户管理人员

## 目录结构

```
project/
├── backend/             # 后端代码目录
├── frontend/            # 前端代码目录
├── database/            # 数据库相关文件
└── documentation/       # 项目文档
```

## 部署步骤

1. 配置Python环境，确保版本为3.8
2. 安装所需依赖，使用`pip`进行安装
3. 配置MySQL数据库，导入数据
4. 使用PyCharm等IDE运行后端项目
5. 构建前端项目，使用Vite进行编译
6. 部署至Web服务器，进行访问测试

## 核心亮点

- 基于Python Django框架，性能稳定
- 高度模块化设计，易于扩展和维护
